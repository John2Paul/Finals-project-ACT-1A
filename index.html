About Assocciate in Computer Technology

The ACT program is a two-year program designed to equip students with the necessary skills and knowledge for careers in the field of computer technology.

The program typically covers fundamental computer programming, basic computation, data structures, network concepts, computer organization, database management systems, and system analysis and design.
